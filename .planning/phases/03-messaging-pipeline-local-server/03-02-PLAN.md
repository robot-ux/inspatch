Implementation details per locked decisions:

**Connection lifecycle (D-01, D-02):**
- On mount: create `new WebSocket(url)` and store in `useRef`.
- On unmount: close WebSocket (cleanup function from useEffect).
- Use `useRef` for the WebSocket instance to prevent re-render issues (per pitfall 2).
- Use empty dependency array `[]` for the connection effect. The `url` should be stable (pass as constant).

**Exponential backoff reconnection (D-04):**
- Backoff starts at 1000ms.
- On disconnect: set status to `'reconnecting'`, schedule reconnect after current backoff delay.
- Double backoff on each attempt: `Math.min(backoff * 2, 30_000)`.
- On successful connection: reset backoff to 1000ms.
- Store backoff in `useRef`, reconnect timeout ID in `useRef`.

**Keepalive ping (D-05):**
- After connection opens: start `setInterval` at 20_000ms that sends `JSON.stringify({ type: "ping" })`.
- After sending ping: start a 5000ms timeout. If no pong received before timeout, treat as disconnected — close the socket, trigger reconnect.
- On receiving `{ type: "pong" }`: clear the pong timeout.
- Clear ping interval and pong timeout on disconnect/cleanup.

**Message validation (D-12):**
- On `ws.onmessage`: `JSON.parse(event.data)`.
- Check if `parsed.type === "pong"` → clear pong timeout, return (don't set lastMessage).
- Otherwise validate with `parseMessage(parsed)` from `@inspatch/shared`.
- If valid: set `lastMessage` state to the validated data.
- If invalid: log warning `console.warn('Invalid message from server:', parsed)` and discard.

**Connection state broadcasting (D-03, COMM-04):**
- Create a helper function `broadcastConnectionState(connected: boolean)`:
  ```typescript
  async function broadcastConnectionState(connected: boolean) {
    await chrome.storage.session.set({
      serverConnected: connected,
      serverPort: parseInt(url.split(':').pop() || '9377', 10),
      connectedAt: connected ? Date.now() : null,
    });
    chrome.runtime.sendMessage({
      type: 'connection_status',
      connected,
    }).catch(() => {});
  }
  ```
- Call with `true` on WebSocket open, `false` on close.

**send function:**
- Takes a `Message` type, `JSON.stringify()` it, and call `ws.send()`.
- Only send if `ws.readyState === WebSocket.OPEN`.
- If not open, log warning.

Do NOT use `WebSocket` from any library — use the browser's native `WebSocket` constructor.
