**Step 1 — Add global styles to style.css** (append after the last .animate-* class, before end of file):

Per D-05 — Custom thin scrollbar (4px, dark colors):
```css
::-webkit-scrollbar {
  width: 4px;
}
::-webkit-scrollbar-track {
  background: var(--ip-bg-primary);
}
::-webkit-scrollbar-thumb {
  background: var(--ip-bg-tertiary);
  border-radius: 2px;
}
::-webkit-scrollbar-thumb:hover {
  background: var(--ip-text-muted);
}
```

Per D-06 — Brand text selection (blue-violet with white text):
```css
::selection {
  background-color: var(--ip-accent-solid);
  color: #FFFFFF;
}
```
Note: `::selection` does not reliably support `background` with gradients across browsers. Use `background-color` with the solid accent fallback `--ip-accent-solid` (#6366F1).

Per D-07 — Focus ring accent glow (replaces browser default outlines):
```css
*:focus-visible {
  outline: none;
  box-shadow: 0 0 0 2px var(--ip-border-accent), var(--ip-shadow-glow-accent);
}
```

Base body style:
```css
body {
  background-color: var(--ip-bg-primary);
  color: var(--ip-text-primary);
  font-family: system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
}
```
